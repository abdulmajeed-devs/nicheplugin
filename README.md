# Domain Data Manager

A professional WordPress plugin for managing domain data with advanced analytics, CSV import/export, and responsive frontend display.

## Features

- **Advanced Analytics:** Track various metrics related to domain data.
- **DA Tracking & Traffic Analysis:** Keep an eye on Domain Authority and website traffic.
- **CSV Data Management:** Easily Bulk Import and Export domain data via CSV formatting.
- **Responsive Frontend Display:** Neatly styled and structured tables for displaying data on your site.
- **Sortable & Mobile-friendly Tables:** Ensure data is easily readable across all devices.
- **Dedicated Admin & Logs:** Full management from the WP Admin Dashboard with integrated logging.

## Installation

1. Download the plugin folder or clone this repository.
2. Upload the `DomainManager` directory to your `/wp-content/plugins/` directory.
3. Keep the folder name as `DomainManager` (or `domain-data-manager`).
4. Activate the plugin through the **Plugins** menu in WordPress.

## Usage

### Admin Dashboard

Once activated, you will see a new menu item in your WordPress admin area for **Domain Data Manager**.
- **Dashboard:** Offers a clean overview of your imported domain data.
- **Upload CSV:** Allows you to upload .csv files populated with your analytical data.
- **Settings:** Configure specific behaviors and display formats.
- **Logs:** View internal operational logs corresponding to plugin activity.

### Frontend Display

Use the provided shortcode (if applicable) or default display partials to render the domain data to your site's frontend visitors.

## Requirements

- **PHP:** >= 7.4
- **WordPress:** >= 5.0 (Tested up to 6.4)
